## **1.如何下载**

如果你不具备Github的相关知识或者从未接触过Git的相关概念，建议你仔细阅读这一段！你可以很轻松的获取下载方式：

> 首先下载就需要一个网址，这个网址其实就是本页面（Github中称为仓库）的地址，也就是：https://github.com/Deyou-design/myshare  如果你经常通过本页面下载，你可以保存这个地址。
> 
> 通过这个地址（见下图一）点击那个绿油油的按钮在弹出的窗口中点击“Download ZIP”即可下载至本地，见图二。
> 
> 经过测试发现Github可能存在网络下载不稳定的情况（虽然几率不大，但毕竟是外网），如果你在下载过程中发现下载速度异常缓慢的，可以尝试从新下载。不同地区的同学下载速度可能不相同。
> 
> 反正我们这里百兆光纤可以达到几M每秒的下载速度... 如果多次尝试还是下载速度非常缓慢的，可能因为Github默认给仓库的带宽太小所致的，建议错峰下载。

如果你具备Github的相关基础使用经验或者本身就是在使用Git的，请你看这一段。

> 通过Git下载和上文中点击“Download ZIP”下载都能获得相同的东西。但是唯一的区别就是点击“Download ZIP”下载只是单纯的文件。
>
>而使用Git的ssh密钥下载的方式可以得到本仓库的一个完整拷贝形式，包含了Git的分布式系统，对于会使用的朋友来说，方便了后期管理。
>
